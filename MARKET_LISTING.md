# VISEA for OpenClaw

## One-Line Pitch

VISEA is a lovable local operations console for a single OpenClaw agent, with live status, efficiency scoring, and optional auto-heal nudges.

## Short Description

Track one OpenClaw agent in real time from a polished local dashboard. VISEA combines live agent status, activity trend, token visibility, and optional auto-heal logic in a single page that stays open beside your daily work.

## Core Value

- See whether the agent is actively working, thinking, idle, overdue, stalled, or down.
- Watch efficiency score and token usage change over time.
- Nudge the agent manually or let auto-heal recover from long silent periods.
- Keep the experience readable, low-noise, and visually distinctive.

## Target User

- Solo builders running one primary OpenClaw agent
- Operators who want a lightweight local control surface
- Teams evaluating a future multi-agent console

## Current Scope

- Single-agent local monitor
- macOS launchd integration
- Local OpenClaw session and gateway inspection
- Optional watchdog and auto-heal helpers

## Known Limits

- macOS-focused startup flow
- Assumes a local OpenClaw installation
- Not yet designed for multi-agent routing or shared team accounts
